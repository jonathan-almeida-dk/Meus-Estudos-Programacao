git init = inicia o git numa pasta escolhida[(criando um arquivo.git) só faz isso uma vez];
git add . = coloca arquivos modificados na "área de preparação";
git commit -m "nome do commit" = cria um ponto salvo do projeto(salva a versão);
git remote add origin [Url do repositório] = Liga sua pasta local ao repositório do GitHub. >> Ex: git remote add origin https://github.com/jonathan-almeida-dk/chatgpt.git <<
git push -u origin main = Envia seus commits para o GitHub.
git pull = baixa no PC a ultima versão do arquivo no GitHub
git log = mostra os commits já feitos
